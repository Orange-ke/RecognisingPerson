## 本地接口设计

安卓通过类`AccessBaiduManager`类里的静态函数与百度进行交互发送请求和处理返回结果

三类发送请求接口

> sendPhotoToMultiRecognize
> sendFaceForMoreInformation
> uploadFaceInform

两类处理返回结果接口

> handleResponseOfDetect
> handleResponseOfSearch
